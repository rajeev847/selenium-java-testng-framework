# Selenium Java TestNG Framework

## Overview

This repository contains Selenium WebDriver automation examples developed using Java and TestNG.

## Technologies Used

* Selenium WebDriver
* Java
* TestNG
* XPath
* Actions Class
* Waits
* Window Handling
* File Upload
* Web Tables

## Covered Concepts

* Implicit Wait
* Explicit Wait
* Fluent Wait
* Drag and Drop
* Window Handling
* File Upload
* Web Table Validation
* TestNG Annotations

## Project Structure

/waits - Selenium Wait examples

/actions - Mouse and keyboard actions

/fileupload - File upload automation

/windowhandling - Multiple window handling

/webtables - Web table validation
